------------------------------------------------------------------------
This is a program for a basic calculator that can compute four basic math functions. These functions are: addition, subtraction, multiplication, and division.
------------------------------------------------------------------------

PROJECT TITLE: Simple-Calculator
PURPOSE OF PROJECT: A simple calculator that takes input from a user and outputs the result.
VERSION or DATE: 02/17/2022
HOW TO START THIS PROJECT: This program was written in Java through BlueJ. The code can be executed in any Java IDE.
AUTHORS: Cole Cagle
USER INSTRUCTIONS: When prompted, enter the first number and then second number (can be decimals) and then select a valid operand for a result to be computed.
